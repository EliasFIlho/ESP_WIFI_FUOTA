◆ read_recv_data_as_string()

esp_err_t read_recv_data_as_string ( httpd_req_t * req,
char * buffer,
size_t buffer_size )

Função para receber requests POST e escrever em um buffer.

Recebe um request, um buffer e o tamanho do buffer. Caso o request seja maior que o buffer a função irá retornar um dado incompleto. Em caso de timeout a função retornará ESP_FAIL. Caso não tenha dados no request a função retornará ESP_FAIL. Caso tudo ocorra sem problemas a função retorna ESP_OK.
